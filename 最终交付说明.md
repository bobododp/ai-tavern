# AI 酒馆最终交付说明

## 当前交付状态

本版本是本地桌面 App 收尾版，默认不内置任何个人 API 地址、模型名或 API Key。

保留内容：
- 一个基础故事模板：`新的故事档案`
- 桌面 App 内置本地代理，默认从 `http://127.0.0.1:8787/api/image` 起步，端口冲突时自动尝试 `8788-8799`
- 空的聊天模型配置
- 空的生图模型配置
- 可继续编辑的故事设置、画廊、生图确认弹窗和世界书导入入口

不包含内容：
- 个人 DeepSeek API 信息
- 个人生图供应商 API 信息
- 真实 API Key
- 用户历史故事数据
- 用户本地生成图片文件

注意：桌面 App 使用独立的 Electron 本地数据，不会读取你浏览器里的旧 `localStorage`。

## 功能状态

- 主页对话：保留。
- 故事板：默认展示基础模板故事。
- 故事封面：故事设置中可点击“生成故事封面”，会先进入绘画提示词确认弹窗，确认后才提交生图。
- 画廊：图片尽量完整预览，减少裁切；支持查看大图、删除、打开文件夹。
- 设置页：聊天模型和生图模型默认清空，用户自行填写。
- 原型运行状态卡：已移除。
- 皮肤系统：已预留 `body[data-skin="classic"]` 和 CSS 变量扩展口。

## 桌面安装包

已采用 Electron 生成 Windows 安装包。

输出文件：
- 安装版：`dist/AI酒馆 Setup 1.0.0.exe`
- 便携版：`dist/AI酒馆 1.0.0.exe`
- 解包测试版：`dist/win-unpacked/AI酒馆.exe`

运行方式：
- 普通用户优先运行安装版。
- 不想安装时可运行便携版。
- 开发检查可运行 `npm run app:dev`。

图片目录：
- 桌面 App 会把生成图片保存到 Electron 用户数据目录下的 `generated-images`。
- 源码目录里的 `generated-images` 不进入安装包。

本次没有做：
- 代码签名
- 自动更新
- 正式账号系统
- 云端同步

## 验证记录

已完成：
- 全部 JS 语法检查通过
- 个人 API 信息扫描通过
- Electron 依赖安装完成
- Electron 打包完成
- 解包版桌面程序烟测通过：内置代理 `HTTP 200 {"ok":true}`

建议最终人工验收：
1. 运行 `dist/AI酒馆 Setup 1.0.0.exe` 安装。
2. 首次打开后，确认设置页 API 地址、模型名、Key 默认为空。
3. 打开故事板，确认只有基础模板故事，没有个人历史数据。
4. 填入聊天模型和生图模型配置后，测试对话与生图。
5. 画廊里查看图片，确认可预览、删除、打开文件夹。
6. 故事设置中点击“生成故事封面”，确认先出现提示词弹窗。
