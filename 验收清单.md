## 页面与导航

- [ ] 直接打开 `index.html`，主页能正常显示。
- [ ] 侧边栏：首页、故事板、画廊、设置都能切换。
- [ ] 亮色/暗色切换正常，刷新后保持。
- [ ] 侧边栏展开/折叠正常，刷新后保持。
- [ ] 页面上没有乱码、`[object Object]`、未绑定的空按钮。

## 故事板

- [ ] 故事卡片标签显示正常：类型、世界观标签、视角、成熟模式符号。
- [ ] 点击“继续”进入对应故事。
- [ ] 点击“故事设置”打开对应故事设置。
- [ ] 删除非唯一故事正常；只剩一个故事时不允许删除。
- [ ] 自定义故事卡片不再出现乱码。

## 对话主流程

- [ ] 行动、对话、描写模式都能发送内容。
- [ ] 观察/画面模式不会直接扣费生图，先进入绘画提示词确认弹窗。
- [ ] 模型回复后正文、选项、右侧状态能同步更新。
- [ ] 复制、编辑、删除、标记重新生成等消息按钮可用。
- [ ] “查看上下文”弹窗内容可读，并且不暴露 API Key。

## 创建故事与故事设置

- [ ] 创建故事弹窗可打开、关闭、创建。
- [ ] AI 完善设定失败时不清空已有输入。
- [ ] 世界书导入后预览、翻译、写入一致。
- [ ] 故事设置里的世界观、主角、NPC、物品、事件、世界书可保存。
- [ ] 成熟模式按钮显示为 `&`，保存后上下文预览能体现对应模式。
- [ ] 修为/能力等字段布局对齐，没有明显错位。

## 生图与画廊

- [ ] 双击 `启动酒馆.bat` 后，本地代理可用。
- [ ] 设置页生图接口、模型、Key、代理地址保存后刷新仍保留。
- [ ] 默认新文件不内置个人 API 地址、模型名或 API Key；只保留本地代理默认地址。
- [ ] 点击生成画面先弹出绘画提示词确认弹窗。
- [ ] 绘画提示词为中文，不包含负面提示词段落。
- [ ] 风格选择和附加风格要求会进入最终提示词。
- [ ] 人物一致性锚点能被带入提示词，但不会替代当前场景剧情。
- [ ] 确认生成后，当前故事上下文出现对应图片消息。
- [ ] 画廊出现图片卡片，点击可看大图。
- [ ] 画廊图片可删除。
- [ ] 打开文件夹按钮能打开 `generated-images` 或对应图片所在目录。
- [ ] 失败卡片显示真实错误信息，不只显示“没有找到图片字段”。

## 收尾判定

- [ ] 代码语法检查全部通过。
- [ ] 主页、故事板、画廊、设置四个页面手动验证通过。
- [ ] 没有明显乱码、空按钮、重复触发、错误链接。
- [ ] 生图流程至少完成一次成功生成和一次失败提示验证。
- [ ] `问题清单.md` 中 P0 已全部处理，P1 已确认是否可接受。
- [ ] 打包方向记录为 Electron / Tauri，中期内置本地代理和图片目录；皮肤系统保留 CSS 变量和 `data-skin` 扩展口。
